Data Table

Tables display sets of data. They can be fully customized.

Features

  • sortable headers
  • pagination - back, forward buttons; total
  • page size selector

Sample Usage

      
    <t-grid
      [data]="productsData"
      [sortable]="true"
      (paginationChange)="performFetch($event)"
      (sortChange)="performSort($event)"
    >
      <t-column 
        [name]="'ID'"
        [property]="'id'" 
        [sortable]="false"
      ></t-column>
      <t-column
        [name]="'Title'"
        [property]="'title'"
        [sortable]="true"
      ></t-column>
      <t-column
        [name]="'Description'"
        [property]="'description'"
        [sortable]="true"
      ></t-column>
      <t-column
        [name]="'Price'"
        [property]="'price'"
        [sortable]="true"
      ></t-column>
      <t-column
        [name]="'Stock'"
        [property]="'stock'"
        [sortable]="true"
      ></t-column>
    </t-grid>
